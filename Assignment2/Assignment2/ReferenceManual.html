Abstract

The anchor tag is used to create a link between pages. The href attribute determines the destination of the link. This tag can be used to create local links, links to other websites, or even bookmarks to different sections of the same page. Other relevant attributes of the anchor tag include target, img, and id.

Details

  • download - Target will be downloaded once the hyperlink is clicked.
  • href - URL of the linked page.
  • hreflang - Specifies the language of the linked document.
  • media - Specifies what media or device the linked document is optimized for.
  • rel - Defines the relationship between the current document and the linked document.
  • target - Specifies where to open the linked document.
  • type - Specifies the media type of the linked document.

Examples

Example for relative link:

<a href="./RelativePageName.html">Link Name</a>

Example for absolute link:

<a href="https://www.google.com">Link Name</a>

Example for adjusting target of link:

<a href="./RelativePageName.html" target=_blank>Link Name</a>
